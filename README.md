# LearnTrack

LearnTrack is a console-based Student & Course Management System built using Core Java. It allows administrators to manage students, courses, and course enrollments entirely in-memory.

## How to Compile and Run
1. Open a terminal and navigate to the `src` directory.
2. Compile all Java files:
   `javac com/airtribe/learntrack/**/*.java`
3. Run the application:
   `java com.airtribe.learntrack.ui.Main`
